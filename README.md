# KnowledgeMate

**Track:** Agents for Good (Education)  
**Capstone Project:** AI Tutor & Learning Assistant

**Project Overview:**  
KnowledgeMate is an AI multi-agent system designed to enhance personalized learning. It helps students by planning learning paths, tutoring in real-time, generating content, and assessing progress automatically. This project demonstrates the use of multi-agent systems, session & memory management, tool integration, and observability.

**Key Features:**
- Multi-agent system: Planner, Tutor, Assessor, Content Generator
- Tool integration: Curriculum Tool, Code Execution, Web Search
- Memory: Short-term and long-term memory storage
- Observability: Logging and evaluation metrics
- Demo-ready Kaggle Notebook

**Submission Contents:**
- `src/` - Full implementation of agents, tools, memory, and evaluation
- `data/` - Sample curriculum, problem bank, and user data
- `notebooks/experiments.ipynb` - Demo & experiments notebook
- `demo_notebook.ipynb` - Submission-ready Kaggle notebook
- `Dockerfile` - Optional deployment
- `video_script.md` - Video script for presentation
